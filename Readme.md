# Installation:

1. Install Docker

`sudo apt-get install docker`

2. Install Docker Compose

`sudo apt-get install docker-compose`

3. Install IoTeX Node[^1]

`./install.sh`


[^1]: You might have to do `chmod +x ./install.sh` prior to executing the script
